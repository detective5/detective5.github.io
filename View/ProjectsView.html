Face Lock

What is Face Lock?
FaceLock is an application that uses the camera on the device to identify the user. Once the user is verified, it then returns a unique code to encrypt/decrypt files on the device for security purposes. FaceLock uses Face++ api for face detection. To learn more about Face++ Click here . For encryption and decryption, FaceLock uses the AES encryption algorithm. It is a advanced algorithm that has adopted by the U.S government.
